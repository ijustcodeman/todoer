# Quellenverzeichnis

Folgende Quellen und Dokumentationen wurden für die Bearbeitung der Aufgabe verwendet:

1.  **Android Developers - Room Persistence Library**
    *   Thema: Implementierung von Entities, DAOs und Datenbank-Instanzen sowie n:m Beziehungen (Junctions).
    *   URL: [https://developer.android.com/training/data-storage/room](https://developer.android.com/training/data-storage/room)

2.  **Material Design - App Bars: Top**
    *   Thema: Styling und Konfiguration der MaterialToolbar und des Popup-Themes.
    *   URL: [https://m2.material.io/components/app-bars-top](https://m2.material.io/components/app-bars-top)

3.  **Android Developers - Save user UI state**
    *   Thema: Nutzung der Preference Library zur Speicherung von Benutzer-Einstellungen wie der Schriftgröße.
    *   URL: [https://developer.android.com/develop/ui/views/components/preferences](https://developer.android.com/develop/ui/views/components/preferences)

4.  **Android Developers - RecyclerView & ItemTouchHelper**
    *   Thema: Implementierung von Listen sowie der "Swipe-to-Delete" Funktionalität und visuellen Anpassungen im Adapter.
    *   URL: [https://developer.android.com/reference/androidx/recyclerview/widget/ItemTouchHelper](https://developer.android.com/reference/androidx/recyclerview/widget/ItemTouchHelper)

5.  **Android Developers - MediaPlayer Overview**
    *   Thema: Abspielen von System-Sounds und Benachrichtigungstönen zur Belohnung des Nutzers.
    *   URL: [https://developer.android.com/guide/topics/media/mediaplayer](https://developer.android.com/guide/topics/media/mediaplayer)

6.  **Android Developers - Menus**
    *   Thema: Erstellung von dynamischen Menüs und Untermenüs für die Sortierfunktion.
    *   URL: [https://developer.android.com/guide/topics/ui/menus](https://developer.android.com/guide/topics/ui/menus)

7.  **Android Developers - Providing backward compatibility with OnBackPressedDispatcher**
    *   Thema: Moderner Ersatz für die veraltete `onBackPressed()` Methode.
    *   URL: [https://developer.android.com/guide/navigation/back-stack/custom-back-navigation](https://developer.android.com/guide/navigation/back-stack/custom-back-navigation)

Für die erstellung von Kommentaren sowie der Verschönerung vom Quellverzeichnis, Wissenslücken und Code Optimierung wurde KI genutzt.
